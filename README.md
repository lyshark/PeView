一款使用C++开发的命令行交互版WindowsPE文件解析器。
<br>

版本：3.0
<br>
发布日期：2021-07-19 13:35

<br>

**打开PE文件:** 使用Open一次性打开文件，只有打开后才可以对其进行其他操作，打开文件需要使用Open命令跟路径。
```C
[Pe View] # Open --path d://Win32Project.exe
[+] 已读入文件
```
